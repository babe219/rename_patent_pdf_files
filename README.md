<http://publicquery.sipo.gov.cn/index.jsp?language=zh_CN>.
pdfminer's tool: pdf2txt.py is the main methods.this run successfully in windows7 .

Requirement；
(1)python 2
(2)pdf2txt.py of pdfminer
